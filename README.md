## 251212 금 ##

로컬호스트(localhost) => 127.0.0.1
: 단어 뜻대로 지역 호스트라는 의미 (현재 본인 pc)

ip란?
: 인터넷상에 연결된 모든 기계(핸드폰, computer, nas 등) 서로를 식별하고 데이터를 통신하기 위해 사용하는 고유한 숫자값
: 한마디로 데이터를 올바른 목적지로 보내기 위한 필수적인 요소 !!

IP주소 종류
- IPv4(32 bits 주소체계 192.168.2.16 과 같은 형태로 표시)
- IPv6 : 128 비트 주소체계

분류방식 : 공인ip, 고정ip, 동적ip, 사설ip
- 공인ip : 전 세계적으로 고유하여 외부 인터넷에서 기기를 식별하는데 사용
- 고정ip : 영구적으로 할당되는 ip주소
- 동적ip : 라우터에서 일시적으로 ip주소를 할당받음. 컴퓨터 재부팅시 ip주소가 바뀔 수 있음.
- 사설ip : 사무실 혹은 가정에서 사용되는 ip

127.0.0.1의 의미
- 본인 pc에 연결된 ip주소가 있듯 localhost에도 짝지어진 ip주소가 127.0.0.1
-> 나 자신을 가리키는 주소

도메인과 ip주소의 관계 
- 웹 브라우저가 도메인과 연결되어 있는 ip주소를 찾아서 이동해줌.

DNS 란 ? Domain Name Service
: 도메인 명을 IP 주소로 변환하는 서비스 
: 네트워크에서 정보를 요청할 때마다 DNS 서버를 통해 도메일 이름과 ip 주소간의 변환작업이 이뤄짐.
: DNS는 인터넷 상에서 정보를 검색하고 서비스를 사용하는데 필수적인 기능으로 이메일, 파일 공유, 문자열 데이터 등 네트워크 서비스를 가능하게 함.

포트란 ?
: 다른 지역으로 갈 수 있는 관문과도 같은 개념
: 외부의 다른 장비와 접속하기 위한 플러그와 같은 것
: 컴퓨터의 여러 프로그램들이 실행되고 있을 때 그 중 어느 프로그램에 접속해야하는지 알려줌 

* compiler vs. interpreter => c code vs. shellscript code

프레임워크 종류
- Spring Boot, Java11, JSP, 톰캣, Nginx

1. jdk11 설치 (jdk : 필요한 라이브러리의 집합)
2. 환경변수 설정 !!

JAVA란
- 객체지향 프로그래밍
- 컴파일러 언어
- jvm 이라는 가상머신 위에서 작동함. 따라서, 운영체제의 종류와 상관없이 자바가 설치된 환경이라면 자바로 개발한 애플리케이션은 정상적으로 작동함 !!
- 자동 메모리 관리 : c/c++과 다르게 객체 생성시 자동으로 메모리 영역을 찾아서 할당하고 안쓰는 객체들은 쓰레기 취급하여 가비지 콜렉션을 실행시켜 자동으로 객   체를 소멸시킴
- jdk : 자바 개발시 필요한 라이브러리들을 모아놓은 도구라고 보면됨. 따라서 자바 애플리케이션을 구동시키기 위해 알맞는 jdk 버전을 설치해야함.

환경변수란
